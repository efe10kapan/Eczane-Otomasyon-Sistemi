# Eczane-Otomasyon-Sistemi
C# Windows Forms kullanılarak geliştirilmiş masaüstü uygulamasıdır. Eczanelerin ilaç stok kayıtlarını, hasta verilerini ve reçete işlemlerini yönetmesini sağlayan, MS Access veritabanı entegreli bir otomasyon çözümüdür.
Kategori,Detaylar
Programlama Dili,C# (.NET Framework)
Kullanıcı Arayüzü,Windows Forms (Desktop)
Veritabanı,Microsoft Access
Mimari Yapı,"CRUD İşlemleri (Ekleme, Listeleme, Güncelleme, Silme)"
Öne Çıkan Özellikler,"İlaç Stok Takibi, Hasta Kayıt Yönetimi ve Kullanıcı Dostu Arayüz"
